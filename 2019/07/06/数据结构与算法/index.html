了解算法及数据结构

数据结构

数据结构是计算机存储、组织数据的方式。数据结构是指相互之间存在一种或多种特定关系的数据元素的集合。通常情况下，精心选择的数据结构可以带来更高的运行或者存储效率的算法。数据结构往往同高效的检索算法和索引技术有关。本质就是数据本身与数据之间的关系。

共有四类基本结构类型：集合、线性结构、树形结构和图形结构。

两种存储结构：顺序和链式存储结构存储结构。

稀疏数组

编写的五子棋程序中，有存盘退出和续上盘的功能

基本介绍

当一个数组中大部分元素为0，或者同一个值的数组时，可以使用稀疏数组来保存改数组。

稀疏数组的处理方法是：

  1. 记录数组一共有几行几列，有多少个不同的值
  2. 把具有不同值的元素的行列及值记录在一个小规模的数组中，从而缩小程序的规模。

单向队列

  1. 队列是一个有序列表，可以用数组或是链表来实现。

  2. 遵循先进先出的原则。

  3. 当rear == maxSize-1时队列满

  4. 当front == rear时队列空

    PS: front指向的是下标的前一位，rear指向的是最后一个下标。

    缺点：数据取出后，空位不能重复使用。所以引入环形队列。

算法

对特定问题求解的步骤的一种描述，它是指令的优先序列，其中一条指令表示一个或多个操作。

他们有五个特性：有穷性，确定性，可行性，输入，输出。

算法设计的要求：正确性，可读性，健壮性，效率与低存储量需求。

算法具有时间复杂度和空间复杂度。（效率度量）
